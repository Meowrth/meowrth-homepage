
Meowrth Homepage

1. This is a simple static homepage for the Meowrth (MEWR) meme coin.
2. To deploy using Netlify:
   - Go to https://app.netlify.com
   - Click 'Add new site' > 'Import an existing project'
   - Connect your Git or drag and drop the unzipped folder
   - Your site will be live on a *.netlify.app domain
